I sistemi che progetto

Sono un ingegnere informatico appassionato di sistemi distribuiti e intelligenza applicata ai dati. Mi occupo principalmente di progettazione e sviluppo di applicativi event-driven per l'anomaly detection in contesti IoT industriali e infrastrutturali.

Lavoro con architetture basate su stream processing e message brokering per analizzare in tempo reale flussi di dati provenienti da sensori e dispositivi connessi, individuando comportamenti anomali prima che diventino problemi critici.

Credo in soluzioni scalabili, osservabili e mantenibili: la complessità del sistema non deve mai ricadere sull'utente finale. Quando non scrivo codice, esploro nuovi approcci algoritmici o contribuisco a discussioni su architetture distribuite.

Progetti selezionati

Sistemi che ho progettato, costruito e messo in produzione.

🔌
Smart Grid Anomaly Monitor

Sistema real-time per il rilevamento di anomalie in reti di distribuzione energetica. Pipeline Kafka + Flink per processare oltre 50.000 eventi/sec da sensori di rete, con modelli ML aggiornati online (online learning).

Python Apache Kafka Apache Flink InfluxDB LSTM
🏭
IIoT Event Processing Platform

Piattaforma modulare per l'Industrial IoT che aggrega telemetria da sensori OPC-UA e Modbus, normalizza i dati tramite schema registry e li espone su topic Kafka per downstream consumers. Deployata su K8s con autoscaling.

Go Kafka OPC-UA Kubernetes Protobuf
📡
Edge Anomaly Detection Framework

Framework leggero per eseguire inferenza di anomaly detection direttamente su dispositivi edge (ARM Cortex-M / Raspberry Pi). Modelli quantizzati con TFLite, soglie adattive calcolate localmente per ridurre latenza e traffico di rete.

C++ Python TensorFlow Lite MQTT Raspberry Pi
📊
Time-Series Observability Stack

Stack di osservabilità per pipeline IoT basato su InfluxDB + Grafana con alerting automatico. Dashboard personalizzate per visualizzare metriche di sistema, anomalie rilevate e KPI operativi. Integrazione con PagerDuty.

InfluxDB Grafana Docker Compose Python Alertmanager
